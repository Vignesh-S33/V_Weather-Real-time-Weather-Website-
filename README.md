V Weather – Real Time Weather Website

---

1. Introduction

The **V Weather Website** is a front-end weather forecasting web application developed using **HTML5, CSS3, and JavaScript**. It is designed to provide users with real-time weather information for any city worldwide using the OpenWeatherMap API.

The project focuses on creating a clean, attractive, and user-friendly interface with live weather updates and basic JavaScript interactivity.

---

2. Objective of the Project

The objectives of this project are:

* To design a real-time weather website
* To fetch live weather data using API integration
* To implement structured layout using HTML
* To apply attractive styling using CSS
* To introduce interactivity using JavaScript
* To demonstrate front-end web development skills

---

3. Project Overview

The V Weather website is a **static front-end project with dynamic API functionality**.

It includes:

* Search input to enter city name
* Live weather data fetching
* Temperature display
* Humidity and wind speed details
* Responsive card-style weather interface

---

4. Technologies Used

| Technology | Purpose |
| ---------- | ------- |
| HTML5 | Structure of the website |
| CSS3 | Styling and layout |
| JavaScript | Interactivity and API handling |
| OpenWeatherMap API | Live weather data |
